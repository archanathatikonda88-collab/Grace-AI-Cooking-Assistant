## ğŸ“¸ SCREENSHOT REQUIREMENTS

### 1. Homepage Screenshot
- **What to capture:** Clean interface with chat area and input fields
- **Browser:** Chrome or Edge, 1920x1080 resolution
- **Elements to show:** 
  - Header with "Grace" title
  - Chat interface area
  - Ingredient input field
  - Difficulty selector
  - "Get Recipes" button

### 2. Recipe Generation Screenshot
- **What to capture:** Chat response with recipe cards displayed
- **Setup:** 
  - Enter "chicken, rice" as ingredients
  - Select "Easy" difficulty
  - Wait for response to load completely
- **Elements to show:**
  - Chat bubble with AI response
  - Recipe cards with images
  - Clean layout separation

### 3. Recipe Detail Screenshot
- **What to capture:** Full recipe detail page
- **Setup:** Click "View Recipe" on any recipe card
- **Elements to show:**
  - Recipe image at top
  - Recipe title and description
  - Ingredients list
  - Step-by-step instructions
  - Back navigation

### 4. Mobile View Screenshot
- **What to capture:** Mobile responsive design
- **Setup:** 
  - Use browser developer tools
  - Set to iPhone 12 Pro (390x844) or similar
  - Show same functionality as desktop
- **Elements to show:**
  - Responsive layout
  - Touch-friendly buttons
  - Proper text sizing

### 5. Architecture Diagram
- **What to create:** Visual representation of system architecture
- **Tools:** Draw.io, Lucidchart, or PowerPoint shapes
- **Elements to include:**
  - Frontend layer
  - Backend layer
  - External APIs
  - Cloud deployment
  - Data flow arrows

### 6. Code Snippet Screenshot
- **What to capture:** Key functions in VS Code
- **Suggested code:**
  - `suggest_recipes()` function from app.py
  - Frontend JavaScript for chat interface
  - CSS for responsive design
- **Setup:** Use dark theme, proper syntax highlighting
